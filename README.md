## 🛠️ Getting Started

1. **Clone the Repository:**
   ```bash

   cd avis-news-repo
   ```
2. **Review Feed Schema:** Inspect how `index.php` aggregates custom RSS streams.  
3. **Local Testing:** Serve via any PHP-enabled web server (Apache, Nginx).
